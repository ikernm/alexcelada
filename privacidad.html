2. Datos que recogemos

A través de los formularios de contacto de este sitio web se recogen los siguientes datos personales:

  • Nombre completo
  • Dirección de correo electrónico
  • Cargo profesional (en formularios específicos)
  • Mensaje o consulta libre

3. Finalidad del tratamiento

Los datos facilitados se utilizan exclusivamente para responder a las consultas recibidas y, en su caso, para el envío de la newsletter semanal si el usuario ha solicitado suscribirse expresamente.

4. Legitimación

La base legal para el tratamiento de sus datos es el consentimiento del interesado, otorgado en el momento de completar y enviar el formulario correspondiente.

5. Conservación de datos

Los datos se conservarán mientras exista una relación activa o hasta que el usuario solicite su supresión.

6. Destinatarios

No se ceden datos a terceros salvo obligación legal. Los formularios utilizan el servicio Web3Forms para el envío de mensajes, sujeto a sus propias políticas de privacidad.

7. Derechos del usuario

El usuario puede ejercer en cualquier momento sus derechos de acceso, rectificación, supresión, oposición y portabilidad escribiendo a info@alexcelada.es.

8. Cookies

Este sitio web no utiliza cookies propias de seguimiento ni analítica de terceros. Si en el futuro se implantaran, esta política se actualizará y se solicitará el consentimiento correspondiente.
